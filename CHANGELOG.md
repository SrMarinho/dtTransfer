# Changelog

## [0.1.0] - 2025-01-01

### Added

- Initial template release
- Declarative YAML workspaces
- Dagster orchestration support
- 4 process types: full, incremental, monthly, unit
- Alembic migrations per workspace
- CLI scaffold: `workspace new`, `entity new`
- Multi-threaded incremental loads
- Telegram error notifications (optional)
- Cookiecutter support
